# Website Pages ‚Äî Formative 1

A small static website (HTML + CSS + images) created as a formative assignment. The site is a multi-page template demonstrating pages for a foundation or community program.

## Quick overview
- Static HTML files for each page.
- Single CSS stylesheet (`style.css`) for layout and styling.
- Images are stored in the `Images/` folder and referenced by the pages.

## Preview locally
- Easiest: open `index.html` in your browser (double-click or "Open with > Browser").
- Optional simple local server (PowerShell / any shell):

  PowerShell:
  ```pwsh
  python -m http.server 8000
  ```
  Then open: http://localhost:8000

  (Requires Python; no install otherwise needed for double-click preview.)
